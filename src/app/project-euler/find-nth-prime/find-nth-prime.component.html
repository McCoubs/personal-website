Find the Nth Prime

Interestingly, there is no currently known way to functionally generate the 'nth prime'. But we can generate an upper bound for the 'nth prime'. So instead, using a prime number sieve (see Sieve of Eratosthenes) and the upper bound, we can sieve the natural numbers up to the upper bound estimate of the nth prime, and then retrieve the nth prime. The result is not very efficient, but it can be improved by using different sieving techniques and generating a more accurate upper bound.
